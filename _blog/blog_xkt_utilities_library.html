Oct 30, 2020

The new double-precision geometry support in xeokit SDK 1.4.8+ includes a version of the xeokit-gltf-to-gltf tool that generates a new version of xeokit's XKT geometry format.

A Utilities Library for Creating XKT Files

xeokit-convert is a new npm library of JavaScript tools for programmatically creating XKT files.

With this library, we can load, or programmatically build, an in-memory document that represents an XKT model.

We can then validate the model, and serialize it to an XKT binary file, which can be loaded into a xeokit viewer using XKTLoaderPlugin.

This library is now used as an external npm dependency by xeokit's xeokit-gltf-to-xkt tool. The library effectively factors out the tool's internal conversion logic into a public npm library that we can reuse in own custom XKT tools.

What can we do with it?


  • Convert a glTF file into an XKT file.

  • Programmatically generate an XKT file.

  • Write custom converters for other source file formats.
