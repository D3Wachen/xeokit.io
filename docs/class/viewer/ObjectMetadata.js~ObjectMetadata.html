    ObjectMetadata

    Metadata for an xeogl.Object within a Viewer's xeogl.Scene.

    This normally represents IFC metadata, but can represent other schemas as well.

    These are kept within the object map and structure tree in Viewer#metadata.

    These only exist for xeogl.Models that were loaded by ModelsPlugins that also loaded metadata.
