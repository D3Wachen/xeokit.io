    Viewer

    The WebGL-based 3D Viewer class at the heart of the xeokit SDK.

    A Viewer wraps a single xeogl.Scene.

    Add Plugins to a Viewer to extend its functionality.

    Save and load the state of a Viewer as JSON with Viewer#getBookmark and Viewer#setBookmark. Installed Plugin instances will also save and load their state to and from the JSON.
