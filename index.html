xeokit - WebGL SDK for BIM & Engineering

xeokit is a software development kit from xeolabs for building Web-based 3D viewers for BIM and engineering. The SDK provides a 3D viewer framework on xeogl, with a library of plugins focused on BIM and CAD. It uses advanced rendering techniques to view and interact with high-detail models in most major browsers, on both desktop and mobile platforms.
