High-Performance 3D Graphics Toolkit
for Web-based BIM and Engineering

xeokit is a software development kit from xeolabs for interacting with high-detail 3D engineering models in the browser.

The SDK provides a high-performance, purpose-built WebGL-based 3D engine with a bundled library of plugins focused on BIM and engineering. The engine uses advanced rendering techniques to view and interact with real-world engineering models in most major browsers, on both desktop and mobile platforms.

xeokit Features

Display your Models in a Browser

Fast loading times and rendering performance, in all major browsers.

  • Interactively display highly-detailed, real-world models.
  • Include metadata to classify and navigate objects.
  • Display models from glTF, OBJ, STL, 3DXML and BIMServer.
  • Works in all major browsers, including mobile.

Programming Toolkit

An open source toolkit geared to engineering visualization on the Web.

  • Purpose-designed WebGL graphics engine.
  • Dynamic scene graph.
  • Move, select, highlight, section and xray objects.
  • Physically-based materials.
  • Clean, modular JavaScript ES6.

Plugins Library

A growing library of plugins to acellerate the development of your application.

  • Sectioning Tools
  • Annotations
  • 2D Storey Plan Views
  • Measurement Tools
  • Navigation Aids
  • BCF Collaboration
