3D Web Programming Toolkit
for AEC Visualization

xeokit is an open source SDK from xeolabs for interacting with high-detail 3D engineering models in the browser.

The xeokit SDK provides a high-performance, purpose-built WebGL-based 3D engine with a bundled library of plugins focused on BIM and engineering. The engine uses advanced rendering techniques to view and interact with real-world engineering models in most major browsers, on both desktop and mobile platforms.
