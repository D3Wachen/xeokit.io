What's included

Licenses include everything in the xeokit Gitub repositories, including:

xeokit-sdk Core SDK and plugins.
xeokit-gltf-to-xkt Multi-platform CLI tool for converting glTF to xeokit's compressed native binary .XKT format. One of the fastest-loading geometry formats for BIM.
xeokit-metadata Multi-platform CLI tool for extracting the structural hierarchy of the building elements within an IFC into the metadata format of the xeokit-sdk.
xeokit-bim-viewer A 3D BIM model viewer for the browser, built on the xeokit SDK. The fastest way to get your IFC models on the Web. Fork the viewer on GitHub and host via GitHub Pages, or download to your own server. Drop in your model files, and you're good to go.
